## Getting Started

1. **Clone the Repository**:
   ```bash

   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd iScrape
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Launch the Application**:
   ```bash
   npm start
   ```
